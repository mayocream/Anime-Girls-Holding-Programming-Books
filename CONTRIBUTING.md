
# Contribution Guidelines

* __NSFW images aren't allowed due to [Github Site Policy](https://docs.github.com/en/site-policy);__

* Pictures can't be related to cryptocurrency or its technologies;
* Images must have descriptive file names in the following format (The file doesn't have to be "png" format, this is just an example):
    Name_Name_Book_Book.png

* Add your images to the folder that best describes the subject matter of the book that the girl is holding. If no existing folder applies, create a new one. The simpler the name, the better;

* All characters must be female presenting (either canonically female or look/act like girls).

* This repo does not accept AI generated images. AI generated images kind of defeat the purpose of this repo since generating a bespoke image of an anime girls holding programming books whenever you want one makes having a repo to catalog them pointless. †


### Notes
† AI generated images can be submitted as long as its clear that the source image was generated for some other purpose and then edited by hand to include a programming book. Including a link to the source image in your PR would help prove this but is not necessary. The key word here is "help", AI generated images tend to have certain look so accumulating too many of those is something the maintainers want to avoid. In the end the reviewers personal aesthetic preference will determine whether or not an AI anime girl gets merged.
